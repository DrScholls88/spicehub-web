# SpiceHub Visual Manifesto: "Organic Precision"

## 1. Visual Direction (High-End Visual Design)
- **Aesthetic**: Brutalist-Lite meets Organic High-End. Sharp geometric borders with soft, fluid color gradients.
- **Palette**: Deep Charcoal (#121212) base, Mint-Cream (#F5FFF5) text, and "Zesty Orange" (#FF6B35) for critical actions.
- **Imagery**: Edge-to-edge photography in the LandingPage hero, using CSS `mask-image` for organic fades.

## 2. The "Feeling" (Design Taste Frontend)
- **Interaction**: Every click should feel "mechanical." Use 150ms `cubic-bezier(0.4, 0, 0.2, 1)` for all transitions.
- **Breathing Room**: Strict 32px padding for sections (Landing/WeekView) to prevent mobile crowding.
- **Micro-Copy**: Clear, human, and slightly witty. (e.g., "Spin the wheel, eat the meal" instead of "Generate Random Recipe").

## 3. Implementation Rules
- **Typography**: /typeset rule: Header-size fluidity. 2.5rem on desktop, 1.8rem on mobile.
- **Borders**: 1px solid #333 for containers; 8px border-radius (Subtle rounding).
- **Motion**: /animate rule: Use "Spring" physics for the MealSpinner—overshoot slightly, then settle.
