# Phase 5 Wizard — key wiring notes

## State (ImportModal.jsx)
- `wizardStep` (1/2/3): auto-advances when `preview` is set
- `collapsedSections`: {ingredients, directions, notes, caption}
- `showSocialPreview`: toggles the live embed behind the collapse card
- `saveDestination`: overrides auto-detect on step 3
- `errorRecovery`: 'browser'|'photo'|'paste'|null — drives recovery buttons

## BrowserAssist fast-path
When `seed._skipPipelineIfCaption` is true AND `seed.capturedCaption` is present,
BrowserAssist skips the full `importFromInstagram` pipeline and goes straight to
`captionToRecipe`. This eliminates the redundant second Apify call.

## Gemini regression fix (recipeParser.js)
`VITE_GEMINI_RESPONSE_SCHEMA=false` (default): NO systemInstruction, NO responseMimeType.
Only when flag=true do we send the schema-aware SYSTEM_INSTRUCTION + responseSchema.
