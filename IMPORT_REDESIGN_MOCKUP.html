What changed & why

1
One scroll region. Header + footer are pinned; only the middle scrolls. Today the whole recipe is one 1500–2500px column inside a ~650px viewport.
2
Collapsed social embed. The 500–700px live IG embed becomes a tap-to-preview card, so the Import button stops getting pushed off-screen.
3
Accordion review sections with count badges + section-level confidence dots; each long section gets its own inner max-height so a 15-step recipe can't blow out the page.
4
Hero image + title overlay. Promotes the wasted 84px thumbnail into the strongest visual asset.
5
Rows simplified to handle + text; move/remove hide behind "⋯". Today each row has 6–7 controls, several sub-44px.
6
"Show original caption" collapsed by default — raw text and structured fields are no longer both expanded at once.
7
Single warm-orange accent. Replaces competing cyan/amber toggles; Meal/Drink shown via tint + icon.
